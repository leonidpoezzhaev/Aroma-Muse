<h1 align="center">Aroma Muse</h1>
<h4 align="center">AromaMuseBot | Ваш персональный аромат по дате рождения</h4>

## Идея
Бот служит для нумерологии.

Вычисляет "аркан" пользователя по дате рождения и полу.

В конце выдает подходящие по аркану и полу ароматы, а также ссылку на их покупку.

## Смысл
Реальный заказ с Авито.

Бот разработан полностью по критериям заказчика.

Сейчас лежит тут и не используется, т.к. не было трафика пользователей.
